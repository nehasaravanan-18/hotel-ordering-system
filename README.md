# Hotel Ordering System (Java)
## Project Description
This is a console-based Hotel Ordering System developed using Core Java. 
The application allows users to select menu items (Veg/Non-Veg), add multiple items to a cart, and generate a structured bill with date, time, and bill number.

## Features
* Menu selection (Veg & Non-Veg)
* Multiple item ordering (Cart system)
* Dynamic bill generation (table format)
* Bill number generation
* Date & time formatted receipt
* Order confirmation system
* Exception handling for invalid inputs
  
## Technologies Used
* Java (Core Java)
* Object-Oriented Programming (OOP)
* Java Collections Framework (ArrayList, LinkedList, Map)
* Exception Handling
* Date & Time API (SimpleDateFormat, Date)

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run the `Main.java` file
4. Follow the console instructions
